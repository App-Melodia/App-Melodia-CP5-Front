# 🎵 Melodia - Sua Música, Sua Forma

Projeto desenvolvido para o Checkpoint 5 da disciplina de Front-end Design - FIAP.

## 📋 Sobre o Projeto
Landing Page do aplicativo de músicas Melodia. O site apresenta os benefícios do app, funcionalidades, depoimentos de usuários e um formulário de contato. Desenvolvido com HTML, CSS e Tailwind CSS.

## 🛠️ Tecnologias
- HTML5
- CSS3
- Tailwind CSS
- Font Awesome
- Google Fonts
- AOS (Animate On Scroll)
